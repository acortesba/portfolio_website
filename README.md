# Alberto Cortés – UX & App Designer | Portfolio
🌐 **Live site:** [albertocortesba.com](https://albertocortesba.com)

🔒 **Source code:** Maintained in a private repository.

A premium, full-stack portfolio experience designed to showcase the intersection of UX/UI excellence and technical implementation. This project features a high-performance 3D showcase, an integrated blog system, and a custom-built administration suite.

## The Vision

This website was built with a "Performance First" philosophy, aiming to provide a seamless, immersive experience while maintaining complete control over every pixel. It serves as both a professional showcase and a playground for experimental web technologies like Three.js and real-time SEO analytics.

## Key Features

- **3D Project Showcase**: Immersive 3D model integration using Three.js for deeper project exploration.
- **Dynamic Theme System**: Six curated themes (Day, Night, Cyberpunk, Monochrome, Blueprint, and Retro) with instant, persistent switching.
- **Integrated Blog**: A full-featured blog platform with rich content blocks and real-time SEO scoring.
- **Interactive Easter Eggs**: Hidden mini-games and interactive elements (Pong, Theme-Hopping, Rock-Breaking) that reward exploration.
- **Privacy & Security**: Secure NDA-protected project access for confidential work samples.
- **Performance Optimized**: Lazy-loading, viewport awareness, and extreme browser-zoom compensation for a consistent look on all devices.

## Core Technologies

- **Frontend**: Vanilla JavaScript (ES6+), Three.js (3D), GSAP (Animations), Editor.js (Rich Text), PixiJS.
- **Backend & Hosting**: Firebase (Firestore, Auth, Storage, Analytics), Hostinger.
- **Design**: Custom CSS systems with retro-futuristic aesthetics and multi-device responsiveness.
